## Configuration and prerequsite
```shell
python3 -m venv /path/to/new/virtual/environment/CS179m
source /path/to/venv/bin/activate
```
Or,
```shell
conda create -n CS179m python=3
conda activate CS179m
```
Then,
```shell
pip install numpy
pip install flask
```
Note that in app.py:
```

```
